# 🚀 Discord Utility Bot

A simple Discord bot made with Python using `discord.py`.

## ✨ Features
- Discord Automod 
- Prefix Only Bot "am!" 
- Supports 27 Commands 
- Ai Auto Moderation 
- Moderation Commands,Logs 
- Open Source 
- In Only One Python Code So You Can Run On Pydroid,Termux Easily! 

## 📌 Run 
1. Git Clone https://github.com/ZensiZapper/AuraFlex-AutoMod-Bot.git
2. pip install discord,aiohttp
3. python main.py 


## 📦 Requirements

- Python 3.10+
- pip

Install dependencies:

```bash
pip install -r requirements.txt
